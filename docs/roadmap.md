# Roadmap

## Version 0.1

Self-hosted AI workspace foundation: auth, providers, chat, agents, explicit memories, MCP configuration, database-backed tasks, feedback, reply intents, Docker, and documentation.

## Version 0.2

Planned email module:

- IMAP ingestion
- SMTP sending
- Gmail OAuth
- Email-linked reply drafts
- Contact-aware context

The Version 0.1 modules keep email out of core chat, tasks, agents, memories, and reply-intent logic.
